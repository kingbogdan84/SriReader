SRIReader Mobile 1.0 (Beta)

=============================================================================
SRIReader Mobile 1.0, es una Aplicación Móvil hecha en HTML5, JS y CSS (SPA)
Sirve para abrir archivos del SRI en formato XML de acuerdo a las
especificaciones de Ecuador y visualizarlos en una interfaz gráfica
entendible y leible

Los archivos que se pueden abrir son:
Facturas, Guias de Remision, Notas de Credito, Notas de Debito y Retenciones


=============================================================================

SRIReader Mobile 1.0 is a Single Page Application / Mobile Application, made in HTML5, JS & CSS3; that can read Ecuadorian IRS XML documents that meet the specifications.
This application allows users to view the data of this files in a friendly way.
